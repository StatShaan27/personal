Overview

1. Binary Operation

  • Definition: a binary operation on a set \(G\) is a map \( *: G \times G \to G\).
  • Meaning: if \(a,b \in G\) then \(a * b \in G\).
  • Examples:
    • On \(\mathbb{Z}\): addition \((a,b)\mapsto a+b\) and multiplication \((a,b)\mapsto ab\) are binary operations.
    • On \(\mathbb{N}\): subtraction is not a binary operation because \(2-5 \notin \mathbb{N}\).

Summary: a binary operation simply combines two elements of the set and returns an element of the same set; closure is the essential requirement.

2. Semigroup

  • Definition: a semigroup is a nonempty set \(G\) with a binary operation that is associative:
    • Associativity: \(\;a(bc)=(ab)c\quad\forall a,b,c\in G.\)
  • Examples:
    • \((\mathbb{Z},+)\) — integers under addition.
    • \((\mathbb{N},\times)\) — natural numbers under multiplication.
  • Non-example: subtraction on integers is not associative:
    • \((5-3)-2 = 0\), but \(5-(3-2)=4\).

Summary: semigroups require closure and associativity only; no identity or inverses are required.

3. Monoid

  • Definition: a monoid is a semigroup that has an identity element \(e\).
  • Identity property: \(ae = ea = a\quad\forall a\in G.\)
  • Examples:
    • \((\mathbb{Z},+)\): identity is \(0\).
    • \((\mathbb{N},\times)\): identity is \(1\).
    • Strings under concatenation: identity is the empty string \(\varepsilon\).

Summary: monoids add a neutral element to the semigroup structure — this often lets you speak meaningfully about powers and repeated application of the operation.

4. Group

  • Definition: a group is a monoid in which every element has an inverse.
  • Inverse property: for each \(a\in G\) there exists \(a^{-1}\in G\) with \(aa^{-1}=a^{-1}a=e.\)
  • Examples:
    • \((\mathbb{Z},+)\): additive inverse of \(n\) is \(-n\).
    • \((\mathbb{Q}\setminus\{0\},\times)\): multiplicative inverse of \(q\) is \(q^{-1}\).
  • Non-example: \((\mathbb{N},+)\) is not a group — most elements lack additive inverses in \(\mathbb{N}\).

Summary: groups are the full package—closure, associativity, identity, and inverses—giving a highly symmetric algebraic structure useful across mathematics.

5. Commutative (Abelian) Groups

  • Definition: the operation is commutative if \(ab=ba\) for all \(a,b\in G\).
  • If a semigroup/monoid/group satisfies commutativity it is called Abelian.
  • Examples and notes:
    • \((\mathbb{Z},+)\) is an Abelian group.
    • The set of invertible \(n\times n\) matrices under multiplication forms the group \(\mathrm{GL}(n,\mathbb{R})\); this group is generally not Abelian.

Summary: commutativity is an additional symmetry that simplifies structure and representation theory; many familiar groups (additive number systems) are Abelian, while many important matrix groups are not.

6. Order of a Group

  • Definition: the order of a group \(G\) is \(|G|\), the number of elements in \(G\).
  • Examples:
    • Finite group: \((\mathbb{Z}_5,+)\) has order \(5\).
    • Infinite group: \((\mathbb{Z},+)\) has infinite order.

Summary: order classifies groups into finite and infinite; many theorems (e.g., Lagrange's theorem) tie element orders and subgroup sizes to the group order.

Theorem 1.2 — Basic Properties of Monoids and Groups

Part 1: Uniqueness of Identity

  • If a monoid \(G\) has two identities \(e_1, e_2\):
    • Since \(e_1\) is identity, \(e_1e_2=e_2\).
    • Since \(e_2\) is identity, \(e_1e_2=e_1\).
    • Hence \(e_1=e_2\).
  • Example: In \((\mathbb{Z},+)\), identity is \(0\). No other element acts as identity.

Summary: identity in a monoid is unique.

Part (i)

  • If \(c \in G\) and \(cc=c\), then \(c=e\).
  • Proof:
    • Multiply both sides on the left by \(c^{-1}\): \(c^{-1}(cc)=c^{-1}c\).
    • This gives \((c^{-1}c)c=ec=c=e\).
  • Example: In \((\mathbb{Z},+)\), if \(c+c=c\), then \(c=0\).

Part (ii) — Cancellation Law

  • If \(ab=ac\), then \(b=c\). If \(ba=ca\), then \(b=c\).
  • Proof (left cancellation):
    • Suppose \(ab=ac\).
    • Multiply both sides by \(a^{-1}\) on the left: \(a^{-1}(ab)=a^{-1}(ac)\).
    • \((a^{-1}a)b=(a^{-1}a)c \implies eb=ec \implies b=c\).
  • Example: In \((\mathbb{Z},+)\), \(5+x=5+y\implies x=y\).

Part (iii) — Uniqueness of Inverse

  • If \(a^{-1}\) and \(b\) are both inverses of \(a\):
    • \(ab=e\) and \(a^{-1}a=e\).
    • Then \(a^{-1}(ab)=(a^{-1}a)b \implies b=a^{-1}\).
  • Hence inverses are unique.

Part (iv)

  • \((a^{-1})^{-1}=a\).
  • Inverse of the inverse brings you back to the original element.

Part (v)

  • \((ab)^{-1}=b^{-1}a^{-1}\).
  • Check:
    • \((ab)(b^{-1}a^{-1})=a(bb^{-1})a^{-1}=aea^{-1}=aa^{-1}=e\).
    • Similarly, \((b^{-1}a^{-1})(ab)=e\).
  • Example: In \((\mathbb{Z},+)\), \((a+b)^{-1}=-(a+b)=(-b)+(-a)\).

Part (vi)

  • Equations \(ax=b\) and \(ya=b\) have unique solutions.
  • Proof:
    • For \(ax=b\): \(x=a^{-1}b\).
    • For \(ya=b\): \(y=ba^{-1}\).
    • Uniqueness follows from cancellation law.

Summary of Theorem 1.2

   Monoid → identity unique
   Group → 
      (i) only identity satisfies cc=c
      (ii) cancellation laws
      (iii) inverses unique
      (iv) inverse of inverse = original
      (v) (ab)^-1 = b^-1 a^-1
      (vi) equations ax=b, ya=b uniquely solvable

Proposition 1.3 — Characterization of Groups

Statement: A semigroup \(G\) is a group iff:

  • (1) There exists \(e \in G\) with \(ea=a\) (left identity).
  • (2) For each \(a\in G\), there exists \(a^{-1}\in G\) with \(a^{-1}a=e\) (left inverse).

Proof (⇒ direction)

  • If \(G\) is a group:
    • It has a two-sided identity, so \(ea=a\).
    • It has two-sided inverses, so left inverses exist.

Proof (⇐ direction)

  • Suppose semigroup has left identity \(e\) and left inverses.
  • Show \(e\) is two-sided identity:
    • For any \(a\), pick \(a^{-1}\).
    • Compute: \(a=(ea)=((a^{-1}a)a)=(a^{-1}(aa))\).
    • By associativity and cancellation, this forces \(ae=a\).
  • Show \(a^{-1}\) is two-sided inverse:
    • From left inverse: \(a^{-1}a=e\).
    • Multiply by \(a^{-1}\) on the right: \((a^{-1}a)a^{-1}=ea^{-1}\).
    • This gives \(a^{-1}(aa^{-1})=a^{-1}\). By cancellation, \(aa^{-1}=e\).
  • Thus \(G\) is a group.

Visualization

  • Theorem 1.2: gives core algebraic properties in groups.
  • Proposition 1.3: a semigroup with left identity and left inverses “upgrades” to a full group automatically.

Proposition 1.4 — Statement

  • Let \(G\) be a semigroup. Then \(G\) is a group iff for all \(a,b\in G\) the equations \(ax=b\) and \(ya=b\) have solutions in \(G\).

Solution (Proof)

(⇒) If \(G\) is a group, then for all \(a,b\in G\) the equations have solutions

  • This direction is immediate because every element \(a\) has an inverse \(a^{-1}\) in a group.
  • Solutions:
    • For \(ax=b\) take \(x=a^{-1}b\).
    • For \(ya=b\) take \(y=ba^{-1}\).
  • These solutions are in fact unique by cancellation.

So the rightward implication is trivial. \(\square\)

(⇐) If both equations are solvable for every \(a,b\), then \(G\) is a group

  • Goal: construct a two-sided identity and two-sided inverses for every element.
    • ```
  • 1. Existence of an identity.
    • Fix arbitrary \(a\in G\). By hypothesis (take \(b=a\)) there exists \(u\in G\) with \[ a u = a. \tag{1} \]
    • Similarly there exists \(v\in G\) with \[ v a = a. \tag{2} \]
    • Compute \(v = v a u\). Using (2) we have \(v a = a\), so \[ v = (v a) u = a u. \] By (1) \(a u = a\). Hence \(v=a\). Combining gives \(u=v\); call this element \(e:=u=v\).
    • Now show \(e\) is a two-sided identity for every \(b\in G\). For arbitrary \(b\) pick \(x\) with \(a x = b\) (possible by hypothesis). Then \[ e b = e(a x) = (e a)x. \] But \(e a = a\) (since \(e=v\) and \(v a = a\)), so \((e a)x = a x = b\). Thus \(e b = b\).
    • A symmetric argument using a solution of \(ya=b\) shows \(b e = b\). Therefore \(e\) is a two-sided identity for all \(b\in G\).
  • 2. Existence of inverses.
    • Fix \(a\in G\). By hypothesis there exists \(s\in G\) with \[ a s = e, \] and there exists \(t\in G\) with \[ t a = e. \]
    • Compute \(s = s a t = e t = t\). (Here we used \(s a = e\) so \(s a t = e t\), and \(e t = t\).)
    • Thus \(s=t\). Call this common element \(a^{-1}:=s\). Then \[ a a^{-1} = e \quad\text{and}\quad a^{-1} a = e, \] so \(a^{-1}\) is a two-sided inverse of \(a\).
  • Since a two-sided identity \(e\) exists and every element has a two-sided inverse, \(G\) is a group. \(\square\)
    • ```

Remark

  • The key idea: solvability for all left- and right-hand linear-type equations lets you manufacture the identity (from the case \(b=a\)) and then manufacture inverses (from the case \(b=e\)).

Summary: the hypothesis that every equation \(ax=b\) and \(ya=b\) is solvable is exactly the condition needed to force the semigroup to admit a global identity and two-sided inverses, i.e. to be a group.

Related Concepts & Immediate Consequences

  • Once a two-sided identity and inverses exist, standard uniqueness and cancellation results follow:
    • Identity is unique.
    • Inverses are unique.
    • Cancellation laws hold.
  • Other equivalent characterizations of groups:
    • Semigroup + two-sided identity + every element has a two-sided inverse.
    • Semigroup + left identity + left inverses (we proved a variant earlier).
    • Semigroup where every left (or every right) translation is bijective — another common characterization.
  • Practical takeaway: verifying solvability of these linear-type equations is often an easier route to proving a structure is a group.

Other Relevant Points / Suggestions

  • Why propositions like 1.4 matter: they let you check fewer or more convenient conditions in practice (e.g., solvability of translation equations) rather than verifying all group axioms directly.
  • Common exercise variants:
    • Show that if every right translation \(R_a:x\mapsto xa\) is bijective, then the semigroup is a group.
    • Write the full Cayley table of \(D_4^*\) using cycle notation and verify associativity and inverses.
    • In \(S_3\) produce explicit noncommuting elements and compute their products both ways.

Closing insight: Proposition 1.4 is a compact, powerful criterion — solvability of the two linear equations for every pair \((a,b)\) is both necessary and sufficient to guarantee the full two-sided group structure.
