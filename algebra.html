Overview

1. Binary Operation

  • Definition: a binary operation on a set \(G\) is a map \( *: G \times G \to G\).
  • Meaning: if \(a,b \in G\) then \(a * b \in G\).
  • Examples:
    • On \(\mathbb{Z}\): addition \((a,b)\mapsto a+b\) and multiplication \((a,b)\mapsto ab\) are binary operations.
    • On \(\mathbb{N}\): subtraction is not a binary operation because \(2-5 \notin \mathbb{N}\).

Summary: a binary operation simply combines two elements of the set and returns an element of the same set; closure is the essential requirement.

2. Semigroup

  • Definition: a semigroup is a nonempty set \(G\) with a binary operation that is associative:
    • Associativity: \(\;a(bc)=(ab)c\quad\forall a,b,c\in G.\)
  • Examples:
    • \((\mathbb{Z},+)\) — integers under addition.
    • \((\mathbb{N},\times)\) — natural numbers under multiplication.
  • Non-example: subtraction on integers is not associative:
    • \((5-3)-2 = 0\), but \(5-(3-2)=4\).

Summary: semigroups require closure and associativity only; no identity or inverses are required.

3. Monoid

  • Definition: a monoid is a semigroup that has an identity element \(e\).
  • Identity property: \(ae = ea = a\quad\forall a\in G.\)
  • Examples:
    • \((\mathbb{Z},+)\): identity is \(0\).
    • \((\mathbb{N},\times)\): identity is \(1\).
    • Strings under concatenation: identity is the empty string \(\varepsilon\).

Summary: monoids add a neutral element to the semigroup structure — this often lets you speak meaningfully about powers and repeated application of the operation.

4. Group

  • Definition: a group is a monoid in which every element has an inverse.
  • Inverse property: for each \(a\in G\) there exists \(a^{-1}\in G\) with \(aa^{-1}=a^{-1}a=e.\)
  • Examples:
    • \((\mathbb{Z},+)\): additive inverse of \(n\) is \(-n\).
    • \((\mathbb{Q}\setminus\{0\},\times)\): multiplicative inverse of \(q\) is \(q^{-1}\).
  • Non-example: \((\mathbb{N},+)\) is not a group — most elements lack additive inverses in \(\mathbb{N}\).

Summary: groups are the full package—closure, associativity, identity, and inverses—giving a highly symmetric algebraic structure useful across mathematics.

5. Commutative (Abelian) Groups

  • Definition: the operation is commutative if \(ab=ba\) for all \(a,b\in G\).
  • If a semigroup/monoid/group satisfies commutativity it is called Abelian.
  • Examples and notes:
    • \((\mathbb{Z},+)\) is an Abelian group.
    • The set of invertible \(n\times n\) matrices under multiplication forms the group \(\mathrm{GL}(n,\mathbb{R})\); this group is generally not Abelian.

Summary: commutativity is an additional symmetry that simplifies structure and representation theory; many familiar groups (additive number systems) are Abelian, while many important matrix groups are not.

6. Order of a Group

  • Definition: the order of a group \(G\) is \(|G|\), the number of elements in \(G\).
  • Examples:
    • Finite group: \((\mathbb{Z}_5,+)\) has order \(5\).
    • Infinite group: \((\mathbb{Z},+)\) has infinite order.

Summary: order classifies groups into finite and infinite; many theorems (e.g., Lagrange's theorem) tie element orders and subgroup sizes to the group order.
